Для запуска проекта вам понадобится сервер с docker и docker-compose

Клонируйте на него репозиторий и запустите скрипт install.sh, он:
- запросит у вас пароли для пользователей root и mysql-user в СУБД;
- создаст необходимые для первоначального развёртывания файлы;
- внесёт изменения в файлы настроек для упрощения и дальнейшего использования;
- запустит docker compose up -d для сборки и запуска проекта.
